If this command is executed without the (player) argument, it will toggle buddha mode on the sender. If it is executed with the (player) argument and the sender has the rcmds.others.buddha permission, this will toggle buddha mode on the given player.

This command supports online name-completion.

Examples

/buddha - Toggles the buddha mode status on the command sender.
/buddha Notch - Toggles Notch's buddha mode status.
