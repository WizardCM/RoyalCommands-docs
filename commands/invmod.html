RoyalCommands Documentation

/invmod

Basics

Modifies other players' inventories.

Usage: /invmod [player]

Aliases: /modinv

Permission Nodes

  • rcmds.invmod
  • rcmds.exempt.invmod

In Depth

/invmod opens [player]'s inventory, allowing for changes to be seen and made in real time.

This command supports online name-completion.

Examples

/invmod Notch - Opens Notch's inventory for live view and modification.
