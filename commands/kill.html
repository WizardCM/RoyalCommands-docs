RoyalCommands Documentation

/kill

Basics

Kills another player.

Usage: /kill [player]

Aliases: /murder

Permission Nodes

  • rcmds.kill
  • rcmds.exempt.kill

In Depth

/kill will immediately set [player]'s health to zero, killing him.

This command supports online name-completion.

Examples

/kill jeb_ - Kills jeb_.
