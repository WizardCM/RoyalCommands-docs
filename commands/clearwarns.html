RoyalCommands Documentation

/clearwarns

Basics

Clears the warnings on a player.

Usage: /clearwarns [player]

Aliases: /removewarns, /clearwarnings, /removewarnings, /rmwarns, /rmwarnings

Permission Nodes

  • rcmds.clearwarns

In Depth

/clearwarns removes all warnings [player] has incurred via /warn.

This command supports online name-completion.

Examples

/clearwarns Notch - Removes all warnings that Notch has.
